# knnAUC
k-nearest neighbors AUC test (R Package: https://github.com/liyistat/knnAUC)


# Recommended knnAUC R package installation
```{r}
devtools::install_github("liyistat/knnAUC");
library(knnAUC);
help(knnAUC)
```
